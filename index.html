Inbuilt Accessibility Deep Dive

Many standard programs and operating systems include powerful, free accessibility features that don't require external software. Explore the panels below to learn about the native tools available in the most common digital environments.

Microsoft (Windows, M365, Edge) Main Accessibility Site â†’
  • Microsoft Read Aloud (M365/Edge): Text-to-speech built directly into Word, Outlook, and the Edge browser for reading emails, documents, and web pages.
  • Microsoft Dictate (M365): Simple, accurate voice-to-text functionality available within all Microsoft 365 applications for hands-free typing.
  • Microsoft Narrator: The native, free screen reader for Windows, providing spoken and braille output for text and controls.
  • Windows Magnifier: Built-in screen magnification tool to enlarge the screen or specific parts, essential for low vision users.
  • OneNote: Note-taking app with integrated **Immersive Reader** (TTS, line focus, simplified view).
  • Microsoft Co-Pilot (Freemium): AI assistant capable of summarizing long documents, restructuring complex text, and aiding in research synthesis, supporting executive functions.
Google Apps & Chrome Main Accessibility Site â†’
  • Chrome Live Captions: Real-time captioning for any audio or video content played within the Chrome browser, greatly benefiting students who are hard of hearing or studying in quiet environments.
  • Chrome Select to Speak (Read Aloud): A Chrome OS feature (also available as an extension) that reads selected text aloud using a high-quality voice.
  • Google Docs/Slides Voice Typing: Hands-free text input directly into Google applications.
  • High Contrast/Dark Mode Extensions: Available for general visual accessibility and reducing visual stress.
Apple (macOS, iOS, Safari) Main Accessibility Site â†’
  • Apple VoiceOver: Apple's powerful, native screen reader for blind and low vision users across all their devices.
  • Apple Voice Control: Advanced dictation and voice command system that allows hands-free control of the operating system.
  • Apple Live Captions (iOS 16+ / macOS Ventura+): System-wide real-time captioning for media, phone calls, and video conferencing.
  • Apple Read and Speak: Includes **Speak Selection** (reads highlighted text) and **Speak Screen** (reads the whole screen) for quick TTS access.
  • macOS Zoom/Display Tools: Highly configurable screen magnification, colour filters, cursor size adjustment, and inverted colours.
  • Safari Reader View: Removes visual distractions (ads, formatting) from web pages, enhancing reading focus.
Firefox & General Browser Features Main Accessibility Site â†’
  • Firefox Reader View: Similar to Safari, this button strips non-essential elements from a webpage for simplified, focused reading.
  • Standard Zoom & Text Resizing: All major browsers (Chrome, Edge, Firefox, Safari) allow users to zoom the entire page or increase text size only (under Settings â†’ Advanced in Safari).
  • Keyboard Shortcuts: All browsers support standard navigation shortcuts. Knowing these can significantly speed up access for users with mobility issues (e.g., Tab to navigate, Spacebar/Page Down to scroll).
